radiod

Stream online radio on Linux, with MPRIS control.

What is it?

radiod is a lightweight background daemon that streams online radio stations on Linux. It speaks MPRIS, so your desktop media keys, KDE Connect, and playerctl all work out of the box. A companion CLI tool lets you control playback from the terminal.

Stations are defined in a simple TOML config file. You can use the orbox: scheme to pull stream URLs from the Online Radio Box directory, or direct: to point at any stream URL you like.

Requirements

These packages are required before radiod will run:

PackageWhy
gstreamerCore multimedia framework
gst-plugins-baseBasic codecs and elements
gst-plugins-goodMP3, AAC, FLAC, PulseAudio
gst-plugins-badHLS, DASH, HTTP live streaming
gst-plugins-uglyAdditional codec support

Install

curl -fsSL https://raw.githubusercontent.com/jonjomckay/radiod/main/install.sh | bash

Or clone and install manually:

git clone https://github.com/jonjomckay/radiod.git
cd radiod
./install.sh              # latest release
./install.sh v0.1.0       # specific version
./install.sh --build      # build from source

Configuration

Create ~/.config/radiod/config.toml:

[[stations]]
name = "BBC Radio 1"
uri = "orbox:uk/bbcradio1"

[[stations]]
name = "BBC Radio 2"
uri = "orbox:uk/bbcradio2"

[[stations]]
name = "My Direct Stream"
uri = "direct:https://example.com/stream.mp3"

[daemon]
volume = 0.8
default_station = "orbox:uk/bbcradio1"
metadata_poll_interval_secs = 30

A commented example lives at config.example.toml in the repo.

Station URIs

SchemeFormatDescription
orbox: orbox:<country>/<alias> Resolved to a stream URL via the Online Radio Box API
direct: direct:<url> Passed directly to GStreamer

Usage

Start and enable the daemon:

systemctl --user enable --now radiod

Control playback:

radiod-ctl play
radiod-ctl pause
radiod-ctl next          # next station
radiod-ctl previous      # previous station
radiod-ctl now-playing   # show current track
radiod-ctl list-stations # list configured stations
radiod-ctl stop
radiod-ctl volume 0.8           # set volume (0.0–1.0)
radiod-ctl set-station <uri>    # switch to station by URI
radiod-ctl status               # playback status summary
radiod-ctl reload               # reload config from disk
radiod-ctl info                 # full MPRIS properties dump

View logs:

journalctl --user -u radiod -f

Desktop media controls, KDE Connect, and playerctl will discover radiod automatically via MPRIS.

Uninstall

systemctl --user disable --now radiod
rm ~/.local/bin/radiod
rm ~/.local/bin/radiod-ctl
rm ${XDG_CONFIG_HOME:-~/.config}/systemd/user/radiod.service
rm ${XDG_DATA_HOME:-~/.local/share}/applications/radiod.desktop
rm ${XDG_DATA_HOME:-~/.local/share}/icons/hicolor/scalable/apps/radiod.svg
systemctl --user daemon-reload

Config and data directories (${XDG_CONFIG_HOME:-~/.config}/radiod, ${XDG_DATA_HOME:-~/.local/share}/radiod) can be removed manually if you like.
